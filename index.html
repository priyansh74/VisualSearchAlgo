Linear Search

Algorithm Steps involved in this algorithm are:
Step 1: Select the first element as the current element.
Step 2: Compare the current element with the target element. If matches, then go to step 5.
Step 3: If there is a next element, then set current element to next element and go to Step 2.
Step 4: Target element not found.
Step 5: Go to Step 6.
Step 6: Target element found and return location.
Step 7: Exit process.

Binary Search

Algorithm Steps involved in this algorithm are:
Step 1: Let min = 0 and max = n-1
Step 2: Guess the average of max and min, rounded down so that it is an integer.
Step 3: If you guessed the number, You found it.
Step 4: If the guess was too low, set min to be one larger then the guess.
Step 5: If the guess was too high, set max to be one smaller then the guess.
Step 6: Go back to step 2.
Step 7: Exit Process
